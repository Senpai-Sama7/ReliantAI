03 / Capabilities
01
AI Document Processing

Every upload runs through Claude. Extracts customer, amounts, dates, terms, document type, and action requirements. Works on any format, any structure, any quality.

02
Configurable Priority Engine

Set your own dollar thresholds. A $500 dispute scores higher than a clean $50k invoice because urgency isn't just dollar value. Thresholds inject directly into the AI system prompt.

03
Escalation Intelligence

Flags what needs human judgment with a specific reason and severity level — not generic alerts. Blocking, warning, or informational. Configurable per your operation's rules.

04
Bilingual Summaries

Every document generates a 3-sentence summary in both English and Spanish simultaneously. Toggle per document. Your Colombia team sees their language without any manual translation step.

05
AI Chat Assistant

Ask your entire document queue natural language questions. "Who is most overdue?" "Draft a follow-up for the Santos account." Only relevant documents inject as context — stays fast, stays accurate.

06
Structured API Output

Every processed document returns typed JSON — ready for your UiPath bots, RPA platforms, or downstream systems to consume. The integration point is a single endpoint call away.

04 / How It Fits Your Stack
STEP 01
Document Arrives

Invoice, dispute, statement, collections notice, email thread. PDF, image, Excel, Word, EML — any format, any structure. Drop it in the dashboard or call the API endpoint directly.

STEP 02
Claude Processes It

Server-side AI analysis extracts all structured fields, assigns priority based on your configured thresholds, identifies escalation conditions, and generates bilingual summaries. Confidence scored on every output.

POST /api/analyze → structured JSON in ~3s
STEP 03
Your Team Sees What Matters

Dashboard surfaces critical items first. Escalation flags tell your team exactly why something needs human judgment. Spanish summaries route to your Colombia team automatically.

STEP 04
Your Bots Take the Clean Data

UiPath, Power Automate, or any RPA platform calls the structured output endpoint. No manual formatting, no ambiguous fields. The document has already been normalized before automation touches it.

GET /api/documents → bot-ready structured payload
05 / Supported Formats
PDF PNG JPG TIFF DOCX XLSX XLS EML TXT CSV

Scanned/image documents processed via OCR. Confidence score included on every output to flag low-quality extractions.

06 / Architecture
// Request flow
Browser → POST /api/analyze (Vercel serverless)
                → Claude Sonnet (document AI)
                → typed JSON response

Chat        → POST /api/chat (Vercel serverless)
                → Claude Haiku (conversational, selective context injection)

// Security
ANTHROPIC_API_KEY → server-side env only. Never reaches client.

// Stack
React 19 · TypeScript · Vite · Tailwind · Vercel · Cloudflare KV
